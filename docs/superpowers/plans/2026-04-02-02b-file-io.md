# File I/O — Implementation Plan (02b)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add workfile loading and saving to the server so documents persist across restarts, with debounced writes triggered by command execution.

**Architecture:** A new `workfile.rs` module handles reading/writing the `.sigil/` directory structure. On startup the server loads a workfile (if configured via `WORKFILE` env var) into the in-memory `Document`. After each command execution, a dirty signal triggers a background tokio task that debounces writes (500ms). The save task acquires the document lock, serializes pages via core's `serialize_page`, and writes JSON files to the workfile directory. A `manifest.json` tracks document metadata and page order.

**Tech Stack:** Rust 1.94.1, Tokio (fs, time, sync), serde_json, sha2 (for content hashing), anyhow

**IMPORTANT:** Your FIRST action before writing ANY code must be to read `CLAUDE.md` in full. Rules in CLAUDE.md take precedence over code in this plan if they conflict. The server crate uses `anyhow` for errors. All document mutations go through `Document::execute`.

```rust
// crates/server/src/workfile.rs
//! Workfile I/O — reading and writing `.sigil/` directory structures.

use std::path::{Path, PathBuf};
use anyhow::{Context, Result, bail};
use serde::{Deserialize, Serialize};
use uuid::Uuid;

use agent_designer_core::{
    Document, Page, PageId, Node, NodeKind, NodeId,
    serialize::{serialize_page, deserialize_page, page_to_serialized, SerializedPage},
    document::DocumentMetadata,
};

/// The workfile manifest — stored as `manifest.json` in the `.sigil/` root.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Manifest {
    pub schema_version: u32,
    pub name: String,
    pub page_order: Vec<Uuid>,
}

impl Manifest {
    /// Creates a manifest from the current document state.
    pub fn from_document(doc: &Document) -> Self {
        Self {
            schema_version: agent_designer_core::CURRENT_SCHEMA_VERSION,
            name: doc.metadata.name.clone(),
            page_order: doc.pages.iter().map(|p| p.id.uuid()).collect(),
        }
    }
}

/// Loads a workfile from a `.sigil/` directory into a Document.
///
/// Reads `manifest.json` for metadata, then loads each page from `pages/`.
/// Returns the populated Document.
pub async fn load_workfile(workfile_path: &Path) -> Result<Document> {
    // Verify the directory exists
    if !tokio::fs::metadata(workfile_path).await?.is_dir() {
        bail!("workfile path is not a directory: {}", workfile_path.display());
    }

    // Read manifest
    let manifest_path = workfile_path.join("manifest.json");
    let manifest_json = tokio::fs::read_to_string(&manifest_path)
        .await
        .context("failed to read manifest.json")?;
    let manifest: Manifest = serde_json::from_str(&manifest_json)
        .context("failed to parse manifest.json")?;

    let mut doc = Document::new(manifest.name.clone());

    // Load pages
    let pages_dir = workfile_path.join("pages");
    if pages_dir.exists() {
        let mut entries = tokio::fs::read_dir(&pages_dir).await?;
        while let Some(entry) = entries.next_entry().await? {
            let path = entry.path();
            if path.extension().is_some_and(|ext| ext == "json") {
                let json = tokio::fs::read_to_string(&path)
                    .await
                    .with_context(|| format!("failed to read page: {}", path.display()))?;
                let serialized_page = deserialize_page(&json)
                    .map_err(|e| anyhow::anyhow!("failed to deserialize {}: {e}", path.display()))?;

                // Reconstruct page and nodes into the document
                load_page_into_document(&mut doc, &serialized_page)?;
            }
        }
    }

    // Reorder pages according to manifest
    reorder_pages(&mut doc, &manifest.page_order);

    tracing::info!(
        "loaded workfile '{}' with {} pages, {} nodes",
        manifest.name,
        doc.pages.len(),
        doc.arena.len()
    );

    Ok(doc)
}

/// Saves the current document state to a `.sigil/` directory.
///
/// Writes `manifest.json` and each page to `pages/{name}.json`.
/// Creates directories if they don't exist.
pub async fn save_workfile(doc: &Document, workfile_path: &Path) -> Result<()> {
    // Ensure directories exist
    let pages_dir = workfile_path.join("pages");
    tokio::fs::create_dir_all(&pages_dir).await?;

    // Write manifest
    let manifest = Manifest::from_document(doc);
    let manifest_json = serde_json::to_string_pretty(&manifest)?;
    tokio::fs::write(workfile_path.join("manifest.json"), &manifest_json).await?;

    // Write each page
    for page in &doc.pages {
        let serialized = page_to_serialized(page, &doc.arena, &doc.transitions)
            .map_err(|e| anyhow::anyhow!("failed to serialize page '{}': {e}", page.name))?;
        let json = serialize_page(&serialized)
            .map_err(|e| anyhow::anyhow!("failed to serialize page '{}': {e}", page.name))?;

        let filename = sanitize_filename(&page.name);
        tokio::fs::write(pages_dir.join(format!("{filename}.json")), &json).await?;
    }

    tracing::debug!(
        "saved workfile '{}' ({} pages)",
        manifest.name,
        doc.pages.len()
    );

    Ok(())
}

/// Reconstructs a page and its nodes from a SerializedPage into the document.
fn load_page_into_document(
    doc: &mut Document,
    serialized: &SerializedPage,
) -> Result<()> {
    let page_id = PageId::new(serialized.id);
    let page = Page::new(page_id, serialized.name.clone());
    doc.add_page(page)
        .map_err(|e| anyhow::anyhow!("failed to add page: {e}"))?;

    // Build a UUID -> NodeId map as we insert nodes
    let mut uuid_to_id = std::collections::HashMap::new();

    // First pass: insert all nodes into the arena
    for snode in &serialized.nodes {
        let node: Node = serde_json::from_value(serde_json::json!({
            "id": [0, 0],
            "uuid": snode.id.to_string(),
            "kind": snode.kind,
            "name": snode.name,
            "parent": null,
            "children": [],
            "transform": snode.transform,
            "style": snode.style,
            "constraints": snode.constraints,
            "grid_placement": snode.grid_placement,
            "visible": snode.visible,
            "locked": snode.locked
        })).map_err(|e| anyhow::anyhow!("failed to deserialize node '{}': {e}", snode.name))?;

        let node_id = doc.arena.insert(node)
            .map_err(|e| anyhow::anyhow!("failed to insert node: {e}"))?;
        uuid_to_id.insert(snode.id, node_id);
    }

    // Second pass: reconstruct parent-child relationships
    for snode in &serialized.nodes {
        let node_id = uuid_to_id[&snode.id];

        if let Some(parent_uuid) = snode.parent {
            if let Some(&parent_id) = uuid_to_id.get(&parent_uuid) {
                agent_designer_core::tree::add_child(&mut doc.arena, parent_id, node_id)
                    .map_err(|e| anyhow::anyhow!("failed to add child: {e}"))?;
            }
        }
    }

    // Third pass: add root nodes to the page
    for snode in &serialized.nodes {
        if snode.parent.is_none() {
            let node_id = uuid_to_id[&snode.id];
            doc.add_root_node_to_page(page_id, node_id)
                .map_err(|e| anyhow::anyhow!("failed to add root node: {e}"))?;
        }
    }

    // Load transitions for this page
    for st in &serialized.transitions {
        // SerializedTransition uses UUIDs — convert source_node to NodeId
        if let Some(&source_id) = uuid_to_id.get(&st.source_node) {
            let transition = agent_designer_core::prototype::Transition {
                id: st.id,
                source_node: source_id,
                target_page: PageId::new(st.target_page),
                target_node: st.target_node.and_then(|uuid| uuid_to_id.get(&uuid).copied()),
                trigger: st.trigger.clone(),
                animation: st.animation.clone(),
            };
            doc.add_transition(transition)
                .map_err(|e| anyhow::anyhow!("failed to add transition: {e}"))?;
        }
    }

    Ok(())
}

/// Reorders pages to match the manifest's page_order.
fn reorder_pages(doc: &mut Document, page_order: &[Uuid]) {
    let order_map: std::collections::HashMap<Uuid, usize> = page_order
        .iter()
        .enumerate()
        .map(|(i, uuid)| (*uuid, i))
        .collect();

    doc.pages.sort_by_key(|p| {
        order_map.get(&p.id.uuid()).copied().unwrap_or(usize::MAX)
    });
}

/// Sanitizes a string for use as a filename.
fn sanitize_filename(name: &str) -> String {
    name.chars()
        .map(|c| if c.is_alphanumeric() || c == '-' || c == '_' { c } else { '_' })
        .collect::<String>()
        .to_lowercase()
}
```

```rust
// crates/server/src/persistence.rs
//! Debounced persistence — saves the document to disk after a quiet period.

use std::path::PathBuf;
use std::sync::Arc;
use std::sync::Mutex;
use tokio::sync::mpsc;
use tokio::time::{Duration, sleep};

use crate::state::SendDocument;
use crate::workfile;

/// Default debounce interval before saving to disk.
pub const SAVE_DEBOUNCE_MS: u64 = 500;

/// Spawns the persistence background task.
///
/// Returns a sender that triggers a save when a signal is sent.
/// The task waits for `SAVE_DEBOUNCE_MS` after the last signal before saving.
pub fn spawn_persistence_task(
    document: Arc<Mutex<SendDocument>>,
    workfile_path: PathBuf,
) -> mpsc::Sender<()> {
    let (tx, mut rx) = mpsc::channel::<()>(16);

    tokio::spawn(async move {
        loop {
            // Wait for the first dirty signal
            if rx.recv().await.is_none() {
                break; // Channel closed, shut down
            }

            // Debounce: keep resetting the timer while signals arrive
            let debounce = Duration::from_millis(SAVE_DEBOUNCE_MS);
            loop {
                tokio::select! {
                    _ = sleep(debounce) => break, // Timer expired, do the save
                    msg = rx.recv() => {
                        if msg.is_none() {
                            return; // Channel closed
                        }
                        // Reset timer by continuing the loop
                    }
                }
            }

            // Acquire lock and save
            let save_result = {
                let doc_guard = match document.lock() {
                    Ok(g) => g,
                    Err(poisoned) => {
                        tracing::error!("document lock poisoned during save, recovering");
                        poisoned.into_inner()
                    }
                };
                workfile::save_workfile(&doc_guard, &workfile_path).await
            };

            match save_result {
                Ok(()) => tracing::debug!("document saved to {}", workfile_path.display()),
                Err(e) => tracing::error!("failed to save document: {e}"),
            }
        }
    });

    tx
}
```

```rust
/// Signals the persistence task that the document has been modified.

        let _ = tx.try_send(());
```
