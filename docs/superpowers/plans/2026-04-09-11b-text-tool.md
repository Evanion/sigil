# Text Tool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add text node creation, inline canvas editing with HTML overlay, canvas text rendering with wrapping/alignment, and typography controls in the properties panel.

**Architecture:** The text tool follows the shape-tool pattern for node creation (click=auto-width, drag=fixed-width), then enters edit mode via an HTML contenteditable overlay positioned over the canvas node. Text is rendered on canvas when not editing. Typography controls in the Design panel use the existing schema-driven panel system. All mutations use the unified `applyOperations` endpoint with `setField` paths (`kind.content`, `kind.text_style.*`).

**Tech Stack:** Rust (core types + FieldOperation), TypeScript/Solid.js (tool, overlay, panel), Canvas 2D (rendering), HTML contenteditable (editing), Vitest (tests)

```rust
use crate::command::FieldOperation;
use crate::document::Document;
use crate::error::CoreError;
use crate::id::NodeId;
use crate::node::*;
use crate::validate::*;

/// Which field of TextStyle to update.
#[derive(Debug, Clone)]
pub enum TextStyleField {
    FontFamily(String),
    FontSize(StyleValue<f64>),
    FontWeight(u16),
    FontStyle(FontStyle),
    LineHeight(StyleValue<f64>),
    LetterSpacing(StyleValue<f64>),
    TextAlign(TextAlign),
    TextDecoration(TextDecoration),
    TextColor(StyleValue<Color>),
}

/// Updates a single field of a text node's TextStyle.
#[derive(Debug)]
pub struct SetTextStyleField {
    pub node_id: NodeId,
    pub field: TextStyleField,
}

impl FieldOperation for SetTextStyleField {
    fn validate(&self, doc: &Document) -> Result<(), CoreError> {
        let node = doc.arena.get(self.node_id)?;
        if !matches!(node.kind, NodeKind::Text { .. }) {
            return Err(CoreError::ValidationError("node is not a text node".into()));
        }
        match &self.field {
            TextStyleField::FontFamily(f) => validate_node_name(f)?,
            TextStyleField::FontSize(sv) => {
                if let StyleValue::Literal { value } = sv {
                    validate_finite(*value)?;
                    if *value < MIN_FONT_SIZE || *value > MAX_FONT_SIZE {
                        return Err(CoreError::ValidationError(
                            format!("font_size {value} out of range [{MIN_FONT_SIZE}, {MAX_FONT_SIZE}]")));
                    }
                }
            }
            TextStyleField::FontWeight(w) => {
                if *w < MIN_FONT_WEIGHT || *w > MAX_FONT_WEIGHT {
                    return Err(CoreError::ValidationError(
                        format!("font_weight {w} out of range [{MIN_FONT_WEIGHT}, {MAX_FONT_WEIGHT}]")));
                }
            }
            TextStyleField::LineHeight(sv) => {
                if let StyleValue::Literal { value } = sv {
                    validate_finite(*value)?;
                    if *value <= 0.0 {
                        return Err(CoreError::ValidationError("line_height must be > 0".into()));
                    }
                }
            }
            TextStyleField::LetterSpacing(sv) => {
                if let StyleValue::Literal { value } = sv {
                    validate_finite(*value)?;
                }
            }
            _ => {} // FontStyle, TextAlign, TextDecoration, TextColor — enum variants are always valid
        }
        Ok(())
    }

    fn apply(&self, doc: &mut Document) -> Result<(), CoreError> {
        let node = doc.arena.get_mut(self.node_id)?;
        if let NodeKind::Text { ref mut text_style, .. } = node.kind {
            match &self.field {
                TextStyleField::FontFamily(v) => text_style.font_family = v.clone(),
                TextStyleField::FontSize(v) => text_style.font_size = v.clone(),
                TextStyleField::FontWeight(v) => text_style.font_weight = *v,
                TextStyleField::FontStyle(v) => text_style.font_style = *v,
                TextStyleField::LineHeight(v) => text_style.line_height = v.clone(),
                TextStyleField::LetterSpacing(v) => text_style.letter_spacing = v.clone(),
                TextStyleField::TextAlign(v) => text_style.text_align = *v,
                TextStyleField::TextDecoration(v) => text_style.text_decoration = *v,
                TextStyleField::TextColor(v) => text_style.text_color = v.clone(),
            }
            Ok(())
        } else {
            Err(CoreError::ValidationError("node is not a text node".into()))
        }
    }
}
```

## Task 5: Frontend — enhance canvas text rendering

**Files:**
- Modify: `frontend/src/canvas/renderer.ts`

- [ ] **Step 1: Replace the simple fillText with multi-line rendering**

```typescript
case "text": {
  const ts = node.kind.text_style;
  const fontStr = buildFontString(ts);
  ctx.font = fontStr;

  // Text color (from text_style, not fill)
  const textColor = ts.text_color.type === "literal"
    ? srgbColorToRgba(ts.text_color.value) ?? "#000000"
    : "#000000";
  ctx.fillStyle = textColor;
  ctx.textBaseline = "alphabetic";

  const lineHeight = ts.line_height.type === "literal" ? ts.line_height.value : 1.5;
  const fontSize = ts.font_size.type === "literal" ? ts.font_size.value : 16;
  const lh = lineHeight * fontSize; // if lineHeight is a multiplier

  const measurement = measureTextLines(ctx, node.kind.content, fontStr,
    node.kind.sizing ?? "auto_width", width, lh);

  for (const line of measurement.lines) {
    let lineX = x;
    if (ts.text_align === "center") lineX = x + (width - line.width) / 2;
    else if (ts.text_align === "right") lineX = x + width - line.width;

    ctx.fillText(line.text, lineX, y + line.y);

    // Decoration
    if (ts.text_decoration === "underline") {
      ctx.beginPath();
      ctx.moveTo(lineX, y + line.y + 2);
      ctx.lineTo(lineX + line.width, y + line.y + 2);
      ctx.strokeStyle = textColor;
      ctx.lineWidth = 1;
      ctx.stroke();
    } else if (ts.text_decoration === "strikethrough") {
      ctx.beginPath();
      const mid = y + line.y - fontSize * 0.3;
      ctx.moveTo(lineX, mid);
      ctx.lineTo(lineX + line.width, mid);
      ctx.strokeStyle = textColor;
      ctx.lineWidth = 1;
      ctx.stroke();
    }
  }
  break;
}
```

  const node = state.nodes[uuid];

function openTextOverlay(uuid: string) {
  if (activeOverlay) commitAndCloseOverlay();

  activeOverlay = createTextOverlay(node, store.viewport(), canvasRef);

function commitAndCloseOverlay() {
  if (!activeOverlay) return;
  const content = activeOverlay.getContent();
  // If empty, delete the node
  if (!content.trim()) { store.deleteNode(editingUuid); }
  else { store.setTextContent(editingUuid, content); }

  activeOverlay = null;
