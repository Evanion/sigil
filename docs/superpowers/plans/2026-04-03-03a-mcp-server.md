# MCP Server Implementation Plan (Plan 03a)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement an MCP server in the `agent-designer-mcp` crate that exposes tools for AI agents to query and mutate design documents, sharing in-memory state with the existing Axum HTTP server.

**Architecture:** The MCP server runs as a separate stdio-based process that shares `AppState` with the HTTP server via `Arc`. It uses the `rmcp` crate (v1.3.0) with the `server` feature, which includes `schemars` for JSON Schema generation. Tools follow the same mutation pattern as GraphQL mutations: acquire document lock → construct command → execute via `Document::execute` → signal dirty → publish event. The MCP crate defines tool parameter/output types with `schemars::JsonSchema` derives and routes all mutations through the core engine.

**Tech Stack:** Rust, `rmcp` 1.3.0 (server feature), `schemars` 1.0 (via rmcp re-export), `agent-designer-core`, `agent-designer-server` (for `AppState`), `tokio`, `serde`/`serde_json`

---

## Scope

This plan implements MVP tools from Spec 03:

**In scope:**
- Document query (info, tree structure, full state)
- Page operations (list, create, rename, delete)
- Node operations (create, delete, rename, set properties, reparent)
- Token operations (list, create, update, delete)
- Component operations (list)
- Undo/redo
- MCP resources (document tree, token list, component list)
- Integration into server binary (stdio transport)

**Deferred:**
- Component creation from selection, instantiation, overrides (needs more complex multi-step flows)
- Transition/prototype operations (lower priority for agent workflows)
- Asset operations (no asset pipeline yet)
- Snapshot/render to image (requires headless rendering)

```rust
use rmcp::model::ErrorCode;

/// Errors that can occur during MCP tool execution.
#[derive(Debug, thiserror::Error)]
pub enum McpToolError {
    #[error("invalid UUID: {0}")]
    InvalidUuid(String),

    #[error("node not found: {0}")]
    NodeNotFound(String),

    #[error("page not found: {0}")]
    PageNotFound(String),

    #[error("token not found: {0}")]
    TokenNotFound(String),

    #[error("core engine error: {0}")]
    CoreError(#[from] agent_designer_core::CoreError),

    #[error("invalid input: {0}")]
    InvalidInput(String),

    #[error("serialization error: {0}")]
    SerializationError(#[from] serde_json::Error),
}

impl McpToolError {
    /// Convert to an `rmcp::McpError` for returning from tool handlers.
    #[must_use]
    pub fn to_mcp_error(&self) -> rmcp::model::McpError {
        let code = match self {
            Self::InvalidUuid(_) | Self::InvalidInput(_) => ErrorCode::INVALID_PARAMS,
            Self::NodeNotFound(_) | Self::PageNotFound(_) | Self::TokenNotFound(_) => {
                ErrorCode::INVALID_PARAMS
            }
            Self::CoreError(_) | Self::SerializationError(_) => ErrorCode::INTERNAL_ERROR,
        };
        rmcp::model::McpError::new(code, self.to_string(), None)
    }
}
```

```rust
//! MCP server implementation.
//!
//! `SigilMcpServer` holds a reference to the shared `AppState` and implements
//! `rmcp::handler::server::ServerHandler` to handle MCP protocol requests.

use rmcp::handler::server::tool::ToolRouter;
use rmcp::handler::server::ServerHandler;
use rmcp::model::{Implementation, ProtocolVersion, ServerCapabilities, ServerInfo};
use rmcp::tool_router;

use agent_designer_server::state::AppState;

/// The Sigil MCP server.
///
/// Shares `AppState` with the HTTP server so both human (frontend) and agent
/// (MCP) edits flow through the same `Document::execute` pipeline.
pub struct SigilMcpServer {
    /// Shared application state — same instance used by the Axum HTTP server.
    state: AppState,
    /// Tool router that dispatches `call_tool` requests to handler functions.
    tool_router: ToolRouter<Self>,
}

impl SigilMcpServer {
    /// Creates a new MCP server backed by the given `AppState`.
    #[must_use]
    pub fn new(state: AppState) -> Self {
        let tool_router = Self::tool_router();
        Self {
            state,
            tool_router,
        }
    }

    /// Returns a reference to the shared `AppState`.
    pub fn state(&self) -> &AppState {
        &self.state
    }
}

/// Acquires the document lock, recovering from mutex poisoning.
///
/// This is the same pattern used in the GraphQL mutation handlers.
pub fn acquire_document_lock(
    state: &AppState,
) -> std::sync::MutexGuard<'_, agent_designer_server::state::SendDocument> {
    match state.document.lock() {
        Ok(guard) => guard,
        Err(poisoned) => {
            tracing::error!("document mutex poisoned, recovering");
            poisoned.into_inner()
        }
    }
}

#[tool_router]
impl SigilMcpServer {}

impl ServerHandler for SigilMcpServer {
    fn get_info(&self) -> ServerInfo {
        ServerInfo {
            protocol_version: ProtocolVersion::V_2025_03_26,
            capabilities: ServerCapabilities::builder()
                .enable_tools()
                .enable_resources()
                .build(),
            server_info: Implementation {
                name: "sigil".to_string(),
                version: crate::version().to_string(),
            },
            instructions: Some(
                "Sigil is a design tool for creating UX designs. \
                 Use tools to query and modify the document. \
                 All mutations support undo/redo."
                    .to_string(),
            ),
        }
    }
}
```

```rust
//! Document query tools — get_document_info, get_document_tree.

use agent_designer_server::state::AppState;

use crate::server::acquire_document_lock;
use crate::types::{
    DocumentInfo, DocumentTree, NodeInfo, PageTree, TransformInfo,
};

/// Builds a `DocumentInfo` from the current document state.
pub fn get_document_info_impl(state: &AppState) -> DocumentInfo {
    let doc = acquire_document_lock(state);
    DocumentInfo {
        name: doc.metadata.name.clone(),
        page_count: doc.pages.len(),
        node_count: doc.arena.len(),
        can_undo: doc.can_undo(),
        can_redo: doc.can_redo(),
    }
}

/// Builds a full document tree from the current document state.
pub fn get_document_tree_impl(state: &AppState) -> DocumentTree {
    let doc = acquire_document_lock(state);
    let pages = doc
        .pages
        .iter()
        .map(|page| {
            let nodes: Vec<NodeInfo> = page
                .root_nodes
                .iter()
                .flat_map(|&root_id| collect_node_tree(&doc, root_id))
                .collect();
            PageTree {
                id: page.id.uuid().to_string(),
                name: page.name.clone(),
                nodes,
            }
        })
        .collect();

    DocumentTree {
        name: doc.metadata.name.clone(),
        pages,
    }
}

/// Recursively collects a node and all its descendants into a flat list.
fn collect_node_tree(
    doc: &agent_designer_core::Document,
    node_id: agent_designer_core::NodeId,
) -> Vec<NodeInfo> {
    let mut result = Vec::new();
    collect_node_tree_inner(doc, node_id, &mut result, 0);
    result
}

/// Maximum tree traversal depth to prevent stack overflow on cyclic references.
const MAX_TREE_DEPTH: usize = 100;

fn collect_node_tree_inner(
    doc: &agent_designer_core::Document,
    node_id: agent_designer_core::NodeId,
    out: &mut Vec<NodeInfo>,
    depth: usize,
) {
    if depth >= MAX_TREE_DEPTH {
        tracing::warn!("max tree depth ({MAX_TREE_DEPTH}) reached, truncating");
        return;
    }

    let Ok(node) = doc.arena.get(node_id) else {
        return;
    };

    let children_uuids: Vec<String> = node
        .children
        .iter()
        .filter_map(|&cid| doc.arena.uuid_of(cid).ok().map(|u| u.to_string()))
        .collect();

    out.push(NodeInfo {
        uuid: node.uuid.to_string(),
        name: node.name.clone(),
        kind: node_kind_to_string(&node.kind),
        visible: node.visible,
        locked: node.locked,
        children: children_uuids,
        transform: TransformInfo {
            x: node.transform.x,
            y: node.transform.y,
            width: node.transform.width,
            height: node.transform.height,
            rotation: node.transform.rotation,
            scale_x: node.transform.scale_x,
            scale_y: node.transform.scale_y,
        },
    });

    for &child_id in &node.children {
        collect_node_tree_inner(doc, child_id, out, depth + 1);
    }
}

/// Converts a `NodeKind` to a simple string label for agents.
pub fn node_kind_to_string(kind: &agent_designer_core::NodeKind) -> String {
    match kind {
        agent_designer_core::NodeKind::Frame { .. } => "frame".to_string(),
        agent_designer_core::NodeKind::Rectangle { .. } => "rectangle".to_string(),
        agent_designer_core::NodeKind::Ellipse { .. } => "ellipse".to_string(),
        agent_designer_core::NodeKind::Path { .. } => "path".to_string(),
        agent_designer_core::NodeKind::Text { .. } => "text".to_string(),
        agent_designer_core::NodeKind::Image { .. } => "image".to_string(),
        agent_designer_core::NodeKind::Group => "group".to_string(),
        agent_designer_core::NodeKind::ComponentInstance { .. } => {
            "component_instance".to_string()
        }
    }
}
```

```rust
pub fn list_pages_impl(state: &AppState) -> Vec<PageInfo> {
    let doc = acquire_document_lock(state);
    doc.pages
        .iter()
        .map(|page| {

                .root_nodes
                .iter()
                .filter_map(|&nid| doc.arena.uuid_of(nid).ok().map(|u| u.to_string()))
                .collect();

                name: page.name.clone(),

/// Creates a new page.
pub fn create_page_impl(state: &AppState, name: &str) -> Result<PageInfo, McpToolError> {
    let page_uuid = uuid::Uuid::new_v4();
    let page_id = PageId::new(page_uuid);
    let page = Page::new(page_id, name.to_string());

    {
        let mut doc = acquire_document_lock(state);

    }

    state.signal_dirty();

    Ok(PageInfo {
        id: page_uuid.to_string(),
        name: name.to_string(),
        root_nodes: vec![],
    })
}

pub fn delete_page_impl(state: &AppState, page_uuid_str: &str) -> Result<MutationResult, McpToolError> {
    let page_uuid: uuid::Uuid = page_uuid_str
        .parse()
        .map_err(|_| McpToolError::InvalidUuid(page_uuid_str.to_string()))?;
    let page_id = PageId::new(page_uuid);

    {
        let mut doc = acquire_document_lock(state);
        let pos = doc
            .pages
            .iter()
            .position(|p| p.id == page_id)
            .ok_or_else(|| McpToolError::PageNotFound(page_uuid_str.to_string()))?;
        doc.pages.remove(pos);
    }

    state.signal_dirty();

    })

        name: new_name.to_string(),

    })
```

```rust
/// Parses a kind string into a `NodeKind`.
fn parse_node_kind(kind: &str) -> Result<NodeKind, McpToolError> {
    match kind {
        "frame" => Ok(NodeKind::Frame { layout: None }),
        "rectangle" => Ok(NodeKind::Rectangle {
            corner_radii: [0.0; 4],
        }),
        "ellipse" => Ok(NodeKind::Ellipse {
            arc_start: 0.0,
            arc_end: std::f64::consts::TAU,
        }),
        "text" => Ok(NodeKind::Text {
            content: String::new(),
            text_style: Default::default(),
        }),
        "group" => Ok(NodeKind::Group),
        "image" => Ok(NodeKind::Image {
            asset_ref: String::new(),
        }),
        other => Err(McpToolError::InvalidInput(format!(
            "unknown node kind: {other}. Valid kinds: frame, rectangle, ellipse, text, group, image"
        ))),
    }
}

/// Creates a new node in the document.
pub fn create_node_impl(
    state: &AppState,
    kind: &str,
    name: &str,
    page_id: Option<&str>,
    parent_uuid: Option<&str>,
    transform: Option<&TransformInput>,
) -> Result<CreateNodeResult, McpToolError> {
    let node_kind = parse_node_kind(kind)?;
    let node_uuid = uuid::Uuid::new_v4();

    let page_id_typed: Option<PageId> = match page_id {
        Some(id_str) => {
            let parsed: uuid::Uuid = id_str
                .parse()
                .map_err(|_| McpToolError::InvalidUuid(id_str.to_string()))?;
            Some(PageId::new(parsed))
        }
        None => None,
    };

    let initial_transform = transform.map(|t| Transform {
        x: t.x,
        y: t.y,
        width: t.width,
        height: t.height,
        rotation: t.rotation,
        scale_x: t.scale_x,
        scale_y: t.scale_y,
    });

    let cmd = CreateNode {
        node_id: NodeId::new(0, 0),
        uuid: node_uuid,
        kind: node_kind,
        name: name.to_string(),
        page_id: page_id_typed,
        initial_transform,
    };

    let node_info = {
        let mut doc = acquire_document_lock(state);
        doc.execute(Box::new(cmd))?;

        let created_id = doc
            .arena
            .id_by_uuid(&node_uuid)
            .ok_or_else(|| McpToolError::NodeNotFound(node_uuid.to_string()))?;

        // If parent_uuid is specified, reparent the node
        if let Some(parent_str) = parent_uuid {
            let parent_parsed: uuid::Uuid = parent_str
                .parse()
                .map_err(|_| McpToolError::InvalidUuid(parent_str.to_string()))?;
            let parent_id = doc
                .arena
                .id_by_uuid(&parent_parsed)
                .ok_or_else(|| McpToolError::NodeNotFound(parent_str.to_string()))?;

            let parent = doc.arena.get(parent_id)?;
            let new_position = parent.children.len();

            let reparent_cmd = agent_designer_core::commands::tree_commands::ReparentNode {
                node_id: created_id,
                new_parent_id: parent_id,
                new_position,
                old_parent_id: None,
                old_position: None,
            };
            doc.execute(Box::new(reparent_cmd))?;
        }

        let node = doc.arena.get(created_id)?;
        let children_uuids: Vec<String> = node
            .children
            .iter()
            .filter_map(|&cid| doc.arena.uuid_of(cid).ok().map(|u| u.to_string()))
            .collect();

        NodeInfo {
            uuid: node_uuid.to_string(),
            name: node.name.clone(),
            kind: node_kind_to_string(&node.kind),
            visible: node.visible,
            locked: node.locked,
            children: children_uuids,
            transform: TransformInfo {
                x: node.transform.x,
                y: node.transform.y,
                width: node.transform.width,
                height: node.transform.height,
                rotation: node.transform.rotation,
                scale_x: node.transform.scale_x,
                scale_y: node.transform.scale_y,
            },
        }
    };

    state.signal_dirty();

    Ok(CreateNodeResult {
        uuid: node_uuid.to_string(),
        node: node_info,
    })
}

/// Deletes a node by UUID.
pub fn delete_node_impl(state: &AppState, uuid_str: &str) -> Result<MutationResult, McpToolError> {
    let parsed_uuid: uuid::Uuid = uuid_str
        .parse()
        .map_err(|_| McpToolError::InvalidUuid(uuid_str.to_string()))?;

    {
        let mut doc = acquire_document_lock(state);
        let node_id = doc
            .arena
            .id_by_uuid(&parsed_uuid)
            .ok_or_else(|| McpToolError::NodeNotFound(uuid_str.to_string()))?;

        let node = doc.arena.get(node_id)?;
        let snapshot = node.clone();
        let parent_id = node.parent;

        let parent_child_index = parent_id.and_then(|pid| {
            doc.arena
                .get(pid)
                .ok()
                .and_then(|parent| parent.children.iter().position(|&id| id == node_id))
        });

        let mut found_page_id: Option<PageId> = None;
        let mut found_page_root_index: Option<usize> = None;
        for page in &doc.pages {
            if let Some(idx) = page.root_nodes.iter().position(|&nid| nid == node_id) {
                found_page_id = Some(page.id);
                found_page_root_index = Some(idx);
                break;
            }
        }

        let cmd = DeleteNode {
            node_id,
            snapshot: Some(snapshot),
            page_id: found_page_id,
            page_root_index: found_page_root_index,
            parent_id,
            parent_child_index,
        };

        doc.execute(Box::new(cmd))?;
    }

    state.signal_dirty();

    Ok(MutationResult {
        success: true,
        message: format!("Node {uuid_str} deleted"),
    })
}
```

```rust
/// Lists all tokens in the document.
pub fn list_tokens_impl(state: &AppState) -> Vec<TokenInfo> {
    let doc = acquire_document_lock(state);
    doc.token_context
        .iter()
        .map(|token| TokenInfo {
            name: token.name().to_string(),
            token_type: format!("{:?}", token.token_type()).to_lowercase(),
            value: serde_json::to_value(token.value()).unwrap_or(serde_json::Value::Null),
            description: token.description().map(String::from),

/// Parses a token type string.
fn parse_token_type(s: &str) -> Result<TokenType, McpToolError> {
    match s {
        "color" => Ok(TokenType::Color),
        "dimension" => Ok(TokenType::Dimension),
        "font_family" => Ok(TokenType::FontFamily),
        "font_weight" => Ok(TokenType::FontWeight),
        "duration" => Ok(TokenType::Duration),
        "cubic_bezier" => Ok(TokenType::CubicBezier),
        "number" => Ok(TokenType::Number),
        "shadow" => Ok(TokenType::Shadow),
        "gradient" => Ok(TokenType::Gradient),
        "typography" => Ok(TokenType::Typography),
        other => Err(McpToolError::InvalidInput(format!(
            "unknown token type: {other}. Valid types: color, dimension, font_family, \
             font_weight, duration, cubic_bezier, number, shadow, gradient, typography"
        ))),
    }
}

/// Creates a new token.
pub fn create_token_impl(
    state: &AppState,
    input: &CreateTokenInput,
) -> Result<TokenInfo, McpToolError> {
    let token_type = parse_token_type(&input.token_type)?;
    let token_value: TokenValue =
        serde_json::from_value(input.value.clone()).map_err(|e| {
            McpToolError::InvalidInput(format!("invalid token value: {e}"))
        })?;

    let token = Token::new(
        TokenId::new(uuid::Uuid::new_v4()),
        input.name.clone(),
        token_value,
        token_type,
        input.description.clone(),
    )
    .map_err(|e| McpToolError::InvalidInput(format!("token validation failed: {e}")))?;

    let cmd = AddToken {
        token: token.clone(),
    };

    {
        let mut doc = acquire_document_lock(state);
        doc.execute(Box::new(cmd))?;
    }

    state.signal_dirty();

    Ok(TokenInfo {
        name: token.name().to_string(),
        token_type: format!("{:?}", token.token_type()).to_lowercase(),
        value: serde_json::to_value(token.value()).unwrap_or(serde_json::Value::Null),
        description: token.description().map(String::from),
    })
}

/// Updates an existing token.
pub fn update_token_impl(
    state: &AppState,
    input: &UpdateTokenInput,
) -> Result<TokenInfo, McpToolError> {
    let token_type = parse_token_type(&input.token_type)?;
    let token_value: TokenValue =
        serde_json::from_value(input.value.clone()).map_err(|e| {
            McpToolError::InvalidInput(format!("invalid token value: {e}"))
        })?;

    let (old_token, new_token) = {
        let doc = acquire_document_lock(state);
        let existing = doc
            .token_context
            .get(&input.name)
            .ok_or_else(|| McpToolError::TokenNotFound(input.name.clone()))?;
        let old_token = existing.clone();

        let new_token = Token::new(
            TokenId::new(uuid::Uuid::new_v4()),
            input.name.clone(),
            token_value,
            token_type,
            input.description.clone(),
        )
        .map_err(|e| McpToolError::InvalidInput(format!("token validation failed: {e}")))?;

        (old_token, new_token)
    };

    let cmd = UpdateToken {
        new_token: new_token.clone(),
        old_token,
    };

    {
        let mut doc = acquire_document_lock(state);
        doc.execute(Box::new(cmd))?;
    }

    state.signal_dirty();

    Ok(TokenInfo {
        name: new_token.name().to_string(),
        token_type: format!("{:?}", new_token.token_type()).to_lowercase(),
        value: serde_json::to_value(new_token.value()).unwrap_or(serde_json::Value::Null),
        description: new_token.description().map(String::from),
    })
}

/// Deletes a token by name.
pub fn delete_token_impl(
    state: &AppState,
    token_name: &str,
) -> Result<MutationResult, McpToolError> {
    let snapshot = {
        let doc = acquire_document_lock(state);
        doc.token_context
            .get(token_name)
            .ok_or_else(|| McpToolError::TokenNotFound(token_name.to_string()))?
            .clone()
    };

    let cmd = RemoveToken {
        token_name: token_name.to_string(),
        snapshot,
    };

    {
        let mut doc = acquire_document_lock(state);
        doc.execute(Box::new(cmd))?;
    }

    state.signal_dirty();

    Ok(MutationResult {
        success: true,
        message: format!("Token '{token_name}' deleted"),
    })
}
```

```rust
/// Undo the last operation.
pub fn undo_impl(state: &AppState) -> Result<UndoRedoResult, McpToolError> {
    let (can_undo, can_redo) = {
        let mut doc = acquire_document_lock(state);
        doc.undo()?;
        (doc.can_undo(), doc.can_redo())
    };

    state.signal_dirty();
```

```rust
/// Reads a resource by URI, returning JSON content.
pub fn read_resource(
    state: &AppState,
    uri: &str,
) -> Result<Vec<ResourceContents>, rmcp::model::McpError> {
    match uri {
        "sigil://document/tree" => {
            let tree = crate::tools::document::get_document_tree_impl(state);
            let json = serde_json::to_string_pretty(&tree)
                .map_err(|e| rmcp::model::McpError::internal(e.to_string(), None))?;
            Ok(vec![ResourceContents::text(json, uri)])
        }
        "sigil://document/tokens" => {
            let tokens = crate::tools::tokens::list_tokens_impl(state);
            let json = serde_json::to_string_pretty(&tokens)
                .map_err(|e| rmcp::model::McpError::internal(e.to_string(), None))?;
            Ok(vec![ResourceContents::text(json, uri)])
        }
        "sigil://document/components" => {
            let components = crate::tools::components::list_components_impl(state);
            let json = serde_json::to_string_pretty(&components)
                .map_err(|e| rmcp::model::McpError::internal(e.to_string(), None))?;
            Ok(vec![ResourceContents::text(json, uri)])
        }
        _ => Err(rmcp::model::McpError::resource_not_found(
            format!("unknown resource: {uri}"),
            None,
        )),
    }
}
```

```rust
    // Spawn MCP server on stdio if requested.
    // This allows agents to connect via stdin/stdout while the HTTP server
    // runs on the configured port for human users.
    let mcp_handle = if std::env::var("MCP_STDIO").is_ok() {
        tracing::info!("starting MCP server on stdio");
        let mcp_state = state.clone();
        Some(tokio::spawn(async move {
            let mcp_server = agent_designer_mcp::server::SigilMcpServer::new(mcp_state);
            let (stdin, stdout) = rmcp::transport::io::stdio();
            match rmcp::serve_server(mcp_server, (stdin, stdout)).await {
                Ok(running) => {
                    if let Err(e) = running.waiting().await {
                        tracing::error!("MCP server error: {e}");
                    }
                }
                Err(e) => {
                    tracing::error!("MCP server failed to start: {e}");
                }
            }
        }))
    } else {
        None
    };
```
